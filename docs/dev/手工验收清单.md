# 手工验收清单

文档用途：用于当前版本对账工具的人工验收  
适用版本：当前本地网页版 `聚天下 + 携程` 对账工具

## 1. 启动服务

1. 在项目根目录执行：

```powershell
uvicorn app.main:app --reload
```

2. 浏览器访问：

```text
http://127.0.0.1:8000
```

3. 预期结果：
- 页面成功打开
- 能看到 `对账月份`
- 能看到 `平台选择`
- 能看到两个上传框
- 能看到 `开始对账` 按钮

## 2. 页面提交流程

1. 选择 `对账月份 = 2026-03`
2. 平台选择 `携程`
3. 上传：
- `test_data/聚天下.xlsx`
- `test_data/携程.xlsx`
4. 点击 `开始对账`

预期结果：
- 页面显示摘要信息
- 页面显示结果表格
- 页面显示双向订单差异检查区
- 页面显示 `导出 Excel` 按钮

## 3. 摘要信息核对

提交成功后，重点检查：

- `对账月份` 是否显示为 `2026-03`
- `平台` 是否显示为 `携程`
- `成功匹配订单数` 是否显示为 `1908`
- `汇总产品数` 是否显示为 `29`
- `被过滤的非当月平台记录数` 是否显示为 `3`
- `聚天下有、第三方当月无` 是否显示为 `172`
- `第三方有、聚天下当月无` 是否显示为 `0`

至少核对以下产品行：

`【即买即用】【线下扫码】东运旅行卢宅景区成人票+卢宅文创产品1份`
- 核销人次：`746`
- 销售额：`48490.00`
- 采购金额：`38792.00`
- 利润：`9698.00`

### 4.2 良渚古城遗址公园成人票（不含观光车）即买即用

- 核销人次：`647`
- 销售额：`34996.50`
- 采购金额：`25998.00`
- 利润：`8998.50`

### 4.3 黄帝陵提前两小时成人票

- 产品名称：
`【提前两小时】黄帝陵景区  成人票`

## 5. 导出功能核对

1. 点击 `导出 Excel`
2. 下载导出的 `.xlsx` 文件
3. 打开文件后检查：
- 工作表名称为 `对账结果`
- A1/B1 显示对账月份
- A2/B2 显示平台
- 第 4 行是结果表头
- 后续行包含对账结果数据

## 6. 订单差异检查核对

提交成功后，重点检查：

- 页面存在 `聚天下有、第三方当月无` 区域
- 页面存在 `第三方有、聚天下当月无` 区域
- 两个差异明细表都包含 `序号` 列
- `聚天下有、第三方当月无` 区域中能看到订单号 `14347370`
- `聚天下有、第三方当月无` 区域第一条明细序号为 `1`
- `第三方有、聚天下当月无` 区域当前显示为空或 `无`

## 7. 验收结论

如果以上步骤全部符合预期，可认为当前版本已达到“可试用”状态。
